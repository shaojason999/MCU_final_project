## 遙控車
使用PIC18F4520作為microcontroller，並利用XBee作為無線模組，搭配一些sensor，做出一台遙控車  
完整報告請看[System Document.pdf](https://github.com/shaojason999/MCU_final_project/blob/master/System%20Document.pdf)
